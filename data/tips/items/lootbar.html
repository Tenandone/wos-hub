셀프충전이란?

셀프충전은 본인이 직접 해당 화면에서 게임 계정에 로그인한 뒤, 게임 내 상점에서 팩(번들)을 직접 교환(구매)하는 방식입니다. 그래서 본계정 확인이 가장 중요합니다.

시작 전에 꼭 확인 (실수 방지)

  • 연동 계정 확인: 게임 계정이 Google/Facebook에 연결되어 있어야 합니다.
  • 로그인 방식: 연동된 구글/페북으로 로그인합니다.
  • 2차인증 필수: 비밀번호만 치고 끝나는 로그인은 2차인증이 아닙니다. 비밀번호 입력 후 “이 접속이 본인 맞냐?” 같은 추가 인증이 나와야 2차인증입니다.

루트바를 이용하려면 2차인증이 사실상 필수입니다. 설정 방법은 인터넷에 안내가 많습니다.
검색 예시: 구글 2단계 인증 설정, 페이스북 2단계 인증 설정

  1. 링크 접속

  2. (신규라면) 추가 쿠폰 받기
    회원가입/로그인 후 루트바 검색창에 #s4qU 입력 → 추가 쿠폰 확인
  3. 원하는 팩(번들) 선택 → 결제
    구매할 팩을 선택하고 결제를 진행합니다.
  4. 결제 완료 후 “충전” 버튼을 꼭 누르기
    결제만 하고 끝이 아닙니다. 반드시 충전을 눌러 셀프충전 화면으로 진입해야 합니다.
  5. 셀프충전 화면에서 로그인
    연동된 구글/페북으로 로그인합니다.
    이때 2차인증이 진행됩니다(비밀번호 입력 후 추가 확인 단계).
  6. 게임 내 상점에서 동일 금액 팩 교환(구매)
    게임이 로딩되면 게임 내 상점에서 결제한 금액과 동일한 팩을 선택해 교환(구매)합니다.
  7. 교환 중 주의
    교환이 끝날 때까지 절대 창을 종료하지 마세요.
  8. 완료 확인 + 차액 환불
    금액을 전부 소진하지 않았다면 차액은 루트바 지갑으로 환불됩니다.
