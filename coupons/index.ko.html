최신 코드

만료는 UTC 00:00 기준으로 판단됩니다. 코드를 복사해 공식 입력 페이지에서 사용하세요.
마지막 업데이트: —

코드가 안 되면: 만료, 이미 사용함, 서버/지역 제한, 오타 가능성이 있습니다.

입력 방법

  1. 게임에서 아바타(좌상단)를 탭합니다.
  2. Player ID를 복사합니다.
  3. 공식 페이지 열기: wos-giftcode.centurygame.com
  4. Player ID + 쿠폰 코드 + 인증 코드를 입력하고 확인합니다.
  5. 게임 내 우편함에서 보상을 수령합니다.

FAQ

쿠폰 코드는 어디에서 입력하나요?
브라우저에서 공식 Gift Code Center를 이용하세요. 일부 안드로이드에서는 설정 → Gift Code 메뉴가 보일 수 있습니다.
만료 기준은 어떻게 되나요?
만료일의 00:00 UTC 기준으로 만료 여부를 판단합니다.
쿠폰이 작동하지 않아요. 왜죠?
만료, 이미 사용함, 서버/지역 제한, 오타 등이 원인일 수 있습니다. 복사 시 공백이 붙지 않았는지 확인해 주세요.
안내: 팬이 만든 페이지입니다. 코드는 예고 없이 변경될 수 있습니다. 입력은 항상 공식 페이지에서 진행하세요.
