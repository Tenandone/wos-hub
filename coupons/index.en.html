FAQ

Where do I redeem codes?
Use the official Gift Code Center in your browser. On some Android devices, you may also see Settings → Gift Code in-game.
When do codes expire?
Expiry is evaluated at 00:00 UTC on the listed expiry date.
My code didn’t work. Why?
Common reasons: expired code, already redeemed, region/server restrictions, or typos. Make sure you copied the code exactly (no extra spaces).
