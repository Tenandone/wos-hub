引き換え方法

  1. ゲームでアバター（左上）をタップします。
  2. Player ID をコピーします。
  3. 公式ページ: wos-giftcode.centurygame.com
  4. Player ID + ギフトコード + 認証コードを入力して確定します。
  5. ゲーム内のメールで報酬を受け取ります。

FAQ

ギフトコードはどこで入力しますか？
ブラウザで公式 Gift Code Center を利用してください。一部のAndroidでは 設定 → Gift Code が表示される場合があります。
期限の判定基準は？
表示されている期限日の 00:00 UTC を基準に判定されます。
コードが使えません。なぜ？
期限切れ、使用済み、サーバー/地域制限、入力ミスなどが原因です。 コピー時に余計な空白が入っていないか確認してください。
注意: ファンサイトです。コードは予告なく変更される場合があります。引き換えは必ず公式ページで行ってください。
